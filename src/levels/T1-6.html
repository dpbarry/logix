Dictionary

  • \( = \)

    Asserts that two objects are equal.

    \( x = y \iff \forall \phi : \phi(x) \leftrightarrow \phi(y) \)

  • \( < \)

    Asserts that a number is strictly lesser than another.

    \( x < y \iff \exist \epsilon > 0 : x + \epsilon = y \)

  • \( > \)

    Asserts that a number is strictly greater than another.

    \( x > y \iff \exist \epsilon > 0 : x - \epsilon = y \)

  • Matrix

    Mathematical structure representing \(n \times m\) entries across \(n\) rows and \(m\) columns.

    • Key controls: use tab to cycle through the entries, the number keys to select numbers, and the following keys for the UI:
