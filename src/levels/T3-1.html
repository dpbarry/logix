3.1

Training

Info

So far we have only seen logical formulas that directly reference concrete numbers or entries. However, this chapter introduces variables, which “stand in” for other entities. An example is \(x = 5\), but this formula does not denote a proposition; that is, it cannot be verified to be true or false because \(x\) is undefined. In order for a formula to be considered a “sentence” (corresponding to a proposition), any variables it has must be bound. This can be accomplished with quantifiers.

There are two main quantifiers: \(\exist\) and \(\forall\). They both rely on a background domain of discourse: the range of entities that a variable might represent. This can be defined in various ways; in Logix you will either see the quantified variable precede a set-inclusion (starting Chapter 4) or, as in this chapter, the variable’s symbol itself will tacitly define the domain. For example, the variables \(i\) and \(j\) will always correspond to the current matrix’s set of row indices \(I\) and column indices \(J\) respectively.

This level introduces the existential quantifier (read aloud as “there exists”), so named because it singles out the existence of some object which fulfills the following predicate. E.g., in the domain of natural numbers \(\exist x : x = 5\) states that there exists a natural number \(x\) such that that number is equal to 5. As it happens, this is a true statement.

This level uses a shorthand relying on the generalization \(\exist x \exist y : P(x,y) \iff \exist x,y : P(x,y)\).

Dictionary

  • \( [i,j] \)

    References the entry at the \(i\)-th row and \(j\)-th column of the level’s matrix. Note that more conventional notation is \(A[i,j]\) or \(A_{i,j}\), but the matrix is elided here for concision.

  • \( = \)

    Asserts that two objects are equal.

    \( x = y \iff \forall \phi : \phi(x) \leftrightarrow \phi(y) \)

  • \( < \)

    Asserts that a number is strictly less than another.

    \( x < y \iff \exist \epsilon > 0 : x + \epsilon = y \)

  • \( > \)

    Asserts that a number is strictly greater than another.

    \( x > y \iff \exist \epsilon > 0 : x - \epsilon = y \)

  • \( \neq \)

    Asserts that two objects are not equal.

    \( x \neq y \iff \exist \phi : \phi(x) \nleftrightarrow \phi(y) \)

  • \( \leq \)

    Asserts that a number is less than or equal to another.

    \( x \leq y \iff (\exist \epsilon > 0 : x + \epsilon = y) \lor (x = y) \)

  • \( \geq \)

    Asserts that a number is greater than or equal to another.

    \( x \geq y \iff (\exist \epsilon > 0 : x - \epsilon = y) \lor (x = y) \)

  • \( \equiv \)

    Asserts that two formulae are logically equivalent, often via syntactic properties.

    \( v(p) = v(q) \vdash p \equiv q \)

  • \( \top \)

    The verum, a constant representing positive truth value.

    \( \top \equiv \textbf{T} \equiv 1 \)

  • \( \bot \)

    The falsum, a constant representing negative truth value.

    \( \bot \equiv \textbf{F} \equiv 0 \)

  • \( \land \)

    Joins two propositions into one which is only true when both of its conjuncts are true.

    \( p \equiv \top, q \equiv \top \vdash p \land q \)

  • \( \lor \)

    Joins two propositions into one which is true when one of its disjuncts is true.

    \( p \equiv \top \vdash p \lor q, q \equiv \top \vdash p \lor q \)

  • \( \lnot \)

    Negates the truth value of a proposition.

    \( p \equiv \bot \vdash \neg p \)

  • \( \to \)

    Asserts that when a proposition is true, so is the other.

    \( p \equiv \bot \vdash p \to q, q \equiv \top \vdash p \to q\)

  • \( \gets \)

    Asserts that a proposition is true when the other is.

    \( q \equiv \bot \vdash p \gets q, p \equiv \top \vdash p \gets q\)

  • \( \leftrightarrow \)

    Asserts that a proposition is true if and only if the other is as well. Also \(\Longleftrightarrow\)

    \( p \equiv \bot \equiv q \vdash p \leftrightarrow q, p \equiv \top \equiv q \vdash p \leftrightarrow q\)

  • \( \ldots \)

    Elides a sequence of terms following a clear pattern. For example, the numbers 1 through 10 could be written as:

    \(1, 2, 3, \ldots, 9, 10 \)

  • \( I,i \)

    The set of row indices in the current level and its corresponding implicitly restricted variable.

    \( \phi(i) \iff (i \in I) \land (\phi(i)) \)

  • \( J,j \)

    The set of column indices in the current level and its corresponding implicitly restricted variable.

    \( \phi(j) \iff (j \in J) \land (\phi(j)) \)

  • \( \exist \)

    Asserts that there exists an object in the domain of discourse which satisfies the following property.

    \( (\exist x \in \set{x_1, x_2, \ldots, x_n} : \phi(x)) \iff (\phi(x_1) \lor \phi(x_2) \lor \cdots \lor \phi(x_n)) \)

  • \( \forall \)

    Asserts that every object in the domain of discourse satisfies the following property.

    \( (\forall x \in \set{x_1, x_2, \ldots, x_n} : \phi(x)) \iff (\phi(x_1) \land \phi(x_2) \land \cdots \land \phi(x_n)) \)

  • Matrix Mathematical structure representing \(n \times m\) entries across \(n\) rows and \(m\) columns.

  • Proposition Logical entity bearing a truth value (true or false); can be expressed with formal expressions.

  • Predicate A relation on \(n\) arguments asserting some sort of property or relationship. When the arguments are provided, it forms a proposition. Examples: \(\textrm{Likes(Alice, Bob)}\), \(\textrm{Prime(7)}\), or \(n = 5\), with the lattermost predicate being in infix position.

  • Relation Chaining Shorthand method in which multiple statements involving infix predicates can be strung together. E.g. \(x < y < z \Longleftrightarrow x < y \land y < z \). Note that this doesn’t imply transitivity when the relation is intransitive.

  • Logical Connective An operator upon logical terms. Despite the name, they can be unary (just one argument; e.g. \(\neg\)) or even nullary (\(\top, \bot\)).

  • Quantifier An operator that introduces a bound variable and defines its upcoming usage in a formula in terms of either an existential or universal relationship to the domain of discourse.

  • Domain of Discourse The set of entities relevant in some sort of formal discussion, to which mentioned variables can be assumed to belong.
